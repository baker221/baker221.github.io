$\sect26.3$ 命题形式和真值表

命题形式的定义

  1. 任何命题变元都是命题形式。 ——此时称为原子命题形式
  2. 若$\alpha$是命题形式，则$(\neg \alpha)$也是命题形式
  3. 若$\alpha$、$\beta$是命题形式，则$(\alpha \vee \beta)$、$(\alpha \wedge \beta)$、$(\alpha \rightarrow \beta)$、$(\alpha \leftrightarrow\beta)$都是命题形式（注意==有括号！！！==）
  4. 只有有限次地应用1-3构成的符号串才是命题形式

3是4不是，没有括号！！！

$((\neg q))$不是，括号多了！

$\neg p$不是，少括号！

如果定义没有括号，则$p\rightarrow q\rightarrow r$是公式，但是有两种构成方法，含义不相同，是不等价的！

约定

  • 非（$\neg$）的优先级高于其他的
  • 省略命题形式最外层括号

一些注记

  • 括号从右往左加

命题形式的BNF（Bacus Normal Form）

  • $\alpha ::=p|(\neg \alpha)|(\alpha \vee \beta)|(\alpha \wedge \beta)|(\alpha \rightarrow \beta) |(\alpha \leftrightarrow \beta)$

每个命题形式都是有限符号串

指派

  • 从${p_1, p_2,…, p_n}$到${0,1}$的一个函数
  • 成真指派
  • 成假指派

  2. 命题常元：之后会涉及

指派

  • 是映射（map）
  • 成真指派/成假指派
    • 不可能有指派求不出来的情况（所有变元值都确定）
    • 指派只可能有0/1之外的值（古典逻辑一个命题只能有0/1两个值）
  • n个变元，有$2^n$种指派
  • 真值表：表示公式在所有指派下的真假
    • 原则上所有子公式都要列出
    • 实际上保证$\alpha$的值不要求错，跳步不要过多即可

命题形式的类型

  • 重(chong)言式/永真式：所有指派均为成真指派
  • 矛盾式/永假式：所有指派均为成假指派
  • 可满足式：存在某个指派为成真指派
    • 在算法中地位很重要：SAT问题
      • 等价描述：一个命题公式是否是可满足式
      • 第一个NP完全问题（Cook-Levin定理）

与哑元的无关性

  • 定理：证明用归纳法

  • 数学中不能靠只管，每一步都有理由（不要显然/易见）

    • 想不到证明方法的：用反证法、归纳法（对$\alpha$由简单到复杂（所包含的联结词个数）归纳）

$\sect26.4$ 联结词的完全集

为什么只考虑五个联结词？

  • 这五个联结词能否表示所有联结词？
  • 这五个联结词是否有多余的？

要回答这个问题，必须回答：

  • 什么是联结词？
  • 什么是一些联结词表示了一个联结词？
  • 什么是联结词的“多余”？

什么是联结词？

  • 联结词确立了复合命题的构造方式

  • 复合命题建立了真假值对应方式

  • 直观的说，联结词就是用来构造复合命题的连词

$f: D^n \to D$ 称为D上n元运算（D上n元函数）

$A^n = {<a_1, a_2, …, a_n>|a_i \in A, 1\leq i \leq n}$（对应一个卡氏积）

$N(p_1, p_2,…,p_n) \ t_1, t_2,…,t_n\ t_i=0\ or\ 1$

称为${0, 1}$上的n元函数（n元真值函数）

  • 每个联结词c确定了一个真值函数$f_c$
  • 每个真值函数也确定了一个联结词

联结词的表示

  • 什么叫“用$\wedge$和$\to$表示$\leftrightarrow$”？

  • 可写为：两者真值表相同

  • 用$c_1, c_2, …, c_k$表示c $\Rightarrow$ 仅用$c_1, c_2, …, c_k$可以构造一个命题$\alpha$与由c构造的命题等价 $\Rightarrow$存在$\alpha$使$\alpha$在任意指派$<t_1, t_2, …, t_n>$下的值恰为$f_c(t_1, t_2, …, t_n)$

证明：${\neg, \vee, \wedge, \to}$是联结词的一个完全集

​ 证：只要证对任k元真值函数f，存在仅含…中联结词的k原命题形式$\alpha$使得$\alpha$在任意指派下的值即为$f(t_1, t_2, …,t_k)$，对k归纳证明（PPT12-13）

$\sect26.5$ 推理形式

  • 推理是逻辑的研究对象
  • 什么是推理形式？
    • 一组前提，一个结论
    • 前提、结论都是命题
  • 什么是正确的推理形式？
    • 直观上正确的推理应保证：若前提正确，则结论也应该正确
    • 有效的推理：若对$\alpha_1, \alpha_2,…,\alpha_n,\beta$中出现的命题变元的任意指派，若$\alpha_1, \alpha_2,…,\alpha_n$都真，则$\beta$亦
    • 否则，称是无效的或不合理的（存在指派使前提都为真，结论却为假）
    • 是否有效与命题形式的排列次序无关
  • 推理有效的充要条件：$(\alpha_1\wedge\alpha_2\wedge…\wedge\alpha_n)\to\beta$是重言式，记为$\alpha_1, \alpha_2,…,\alpha_n\vDash \beta$
    • （同一符号，从右向左算）
    • 意义：
      • 推理形式的有效性与命题形式的永真性可以互相化约
      • 今后将证明带前提的证明系统和重言式证明系统，并证明它们的等价性
  • 一些有效推理

$\sect 26.8$ N与P的等价性

等价性：$\Gamma\vdash_\mathrm{P}\alpha$当且仅当$\Gamma\vdash_\mathrm{N}\alpha$（更弱的一种表现：$\vdash_\mathrm{P}\alpha \Leftrightarrow \vdash_\mathrm{N} \alpha$）

如何处理N与P连接词不同的问题？

  • 可以扩展P的连接词，但是实际上是将那些连接词在P中看成另一个东西的简写

当$\Gamma$为无限公式集时，对于P可能成立，但对于N不可能成立！

  • 解决方法：若$\Sigma\vdash_\mathrm{P}\alpha$，则存在$\Sigma$的有限子集$\Sigma_0\subseteq\Sigma$，使得$\Sigma_0\vdash_\mathrm{P}\alpha$。（因为证明过程是有限的，故在证明过程中用到的前提必定是有限的）
